# Customer Retention Analysis using SQL Server

## Project Overview

This project analyzes customer purchasing behavior for an e-commerce company using SQL Server. The objective is to identify customer purchasing patterns, measure customer retention, and generate business insights that can help improve customer loyalty and business growth.

---

## Business Problem

An e-commerce company observed that a significant number of customers make only one purchase and never return. The management team wants to understand customer purchasing behavior, identify repeat customers, and measure customer retention using SQL.

---

## Objectives

- Analyze customer purchase frequency.
- Identify repeat and one-time customers.
- Measure customer retention.
- Identify high-value customers.
- Generate actionable business insights using SQL.

---

## Dataset

The project consists of four tables:

| Table | Description |
|--------|-------------|
| Customers | Stores customer information |
| Orders | Stores order details |
| Products | Stores product information |
| Order_Items | Stores products purchased in each order |
